# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), which sets a clear standard for behavior in open-source communities. Read the full text at that link.

## In short

Be respectful. Assume good intent. Disagree on ideas, not on people.

## Scope

This applies to everything connected to the project — issues, pull requests, discussions, commit messages, and any direct communication about Junction.

## Reporting

If you experience or witness behavior that violates the Code of Conduct, email the maintainer (see `git log` for contact). Reports are kept confidential. We will respond, acknowledge receipt, and decide on action privately before anything public happens.

If the maintainer is the source of the problem, escalate to GitHub Support — they can intervene on the platform.

## Enforcement

The maintainer is responsible for clarifying and enforcing standards. Consequences range from a private clarification, to a public warning, to a temporary or permanent ban from project spaces, depending on severity. The Contributor Covenant's [Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) describe the standard ladder we follow.
